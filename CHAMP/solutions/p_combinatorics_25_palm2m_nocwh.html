On a chess board, two rooks are placed peacefully if they are not on the same row or column. For an n x n chess board, find the number of ways that n rooks can be placed peacefully (i.e., any two are placed peacefully), as an expression of n.

1. Place rooks row by row. How many positions are possible for the rook in each row?
The number of ways to place $n$ rooks on an $n\times n$ chess board peacefully is $\binom{n(n-1)}{n-1}$.

To see this, we can place the rooks row by row. In the first row, we have $n$ choices for where to place the first rook. For the second row, we only have $n-1$ choices left, since the first rook cannot be in the same row or column. So far, there are $n(n-1)$ possible placements.

For the third row, we only have $n-2$ choices left, since the first two rooks cannot be in the same row or column. Continuing in this way, for the $k$th row, we only have $n-k+1$ choices left, since the first $k-1$ rooks cannot be in the same row or column.

Therefore, the total number of possible placements is $$n(n-1)(n-2)\dots(n-k+1) = \binom{n(n-1)}{n-1}.$$
