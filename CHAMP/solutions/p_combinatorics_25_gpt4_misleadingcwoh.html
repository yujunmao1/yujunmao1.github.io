On a chess board, two rooks are placed peacefully if they are not on the same row or column. For an n x n chess board, find the number of ways that n rooks can be placed peacefully (i.e., any two are placed peacefully), as an expression of n.

The problem is asking for the number of ways to place n rooks on an n x n chess board such that no two rooks are in the same row or column. This is equivalent to asking for the number of permutations of n items, which is simply n!.

Here's why: 

For the first rook, there are n choices of squares (since there are n rows and n columns). Once the first rook is placed, there are (n-1) choices for the second rook (since one row and one column are now off limits). For the third rook, there are (n-2) choices, and so on, until for the last rook, there is just 1 choice left. 

So the total number of ways to place the rooks is n * (n-1) * (n-2) * ... * 1, which is the definition of n factorial, or n!.

So the answer to the problem is n!.
