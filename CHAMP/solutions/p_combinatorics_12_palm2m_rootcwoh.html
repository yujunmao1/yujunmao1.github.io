Along a one-way street there are n parking lots. One-by-one n cars numbered 1 to n enter the street. Each driver i heads to his favorite parking lot a_i, and, if it is free, he occupies it. Otherwise, he continues to the next free lot and occupies it. But if all succeeding lots are occupied, he leaves for good. How many sequences {a_1, ..., a_n} are such that every driver can park, as an expression of n?

The number of ways to park n cars in n parking lots is given by the following formula:

$$\binom{n+n-1}{n-1}$$

This can be shown by considering the following algorithm:

1. Start with all n parking lots empty.
2. For each car, in order, do the following:
    * If the car's favorite parking lot is empty, park the car there.
    * Otherwise, if the car's favorite parking lot is occupied, park the car in the next empty parking lot.
    * Otherwise, the car cannot park and the algorithm terminates.

The number of ways to park the cars is the number of times the algorithm terminates successfully. This is equal to the number of ways to choose n-1 parking lots to park the cars in, since the first parking lot is always empty. This is given by the following formula:

$$\binom{n+n-1}{n-1}$$

For example, if there are 3 cars and 3 parking lots, the possible sequences of parking are:

1. 1-2-3
2. 1-3-2
3. 2-1-3
4. 2-3-1
5. 3-1-2
6. 3-2-1

In each of these sequences, all three cars are able to park.

The following is a Python implementation of the algorithm:

```python
def park_cars(n):
  """Parks n cars in n parking lots.

  Args:
    n: The number of cars.

  Returns:
    A list of the parking slots for each car.
  """
  slots = [0] * n
  for i in range(n):
    favorite_slot = i
    while slots[favorite_slot] != 0:
      favorite_slot = (favorite_slot + 1) % n
    slots[favorite_slot] = 1
  return slots
```
