On a chess board, two rooks are placed peacefully if they are not on the same row or column. For an n x n chess board, find the number of ways that n rooks can be placed peacefully (i.e., any two are placed peacefully), as an expression of n.

You may find the above concept helpful.
This problem is a classic example of permutations. 

In this case, each rook must be in a different row and column. This is equivalent to arranging the n rooks in n positions, which is a permutation problem.

For the first rook, there are n possible positions (one in each row). After placing the first rook, there are (n-1) positions left for the second rook. For the third rook, there are (n-2) positions left, and so on. 

So, the total number of ways to place n rooks peacefully on an n x n chess board is:

n * (n-1) * (n-2) * ... * 2 * 1

This is the definition of n factorial, denoted as n!.

So, the answer is n!.
