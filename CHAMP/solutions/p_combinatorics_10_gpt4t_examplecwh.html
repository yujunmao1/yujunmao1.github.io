This problem is a classic example of a combinatorial problem that can be solved using the "stars and bars" method. The problem is equivalent to finding the number of ways to partition the number n into a sequence of positive integers.

To visualize this, imagine you have n stars in a row, and you want to divide them into groups (where each group represents a number in the sequence). The dividers will be placed in the gaps between the stars to separate them into groups.

Since we have n stars, there are n - 1 gaps between them. To form a sequence of positive integers that sum up to n, we can place dividers in these gaps. The number of dividers will be one less than the number of terms in the sequence because the dividers separate the terms.

For example, if n = 5, and we want to create sequences like (1, 4), (2, 3), (1, 1, 3), etc., we can represent these with stars and bars as follows:

- (1, 4) can be represented as: * | ****
- (2, 3) can be represented as: ** | ***
- (1, 1, 3) can be represented as: * | * | ***

Now, the problem is reduced to determining in how many ways we can insert dividers into the n - 1 gaps. Since we want sequences of positive integers, we cannot place more than one divider in the same gap (as that would create a group with zero stars, which is not allowed).

The number of ways to place k dividers into n - 1 gaps is given by the binomial coefficient "n - 1 choose k," which is written as C(n - 1, k). However, we don't have a fixed number of dividers; we can have anywhere from 0 to n - 1 dividers, depending on how many terms we want in the sequence.

Therefore, we need to sum over all possible numbers of dividers from 0 to n - 1. But there's a simpler way to think about this: we can consider each sequence as a way to place n - 1 dividers among n - 1 gaps, where some gaps may not get a divider at all.

This is equivalent to choosing n - 1 objects from a set of 2n - 2 objects (n - 1 stars and n - 1 gaps), which is given by the binomial coefficient C(2n - 2, n - 1).

So, the number of sequences of positive integers that sum to n is:

C(2n - 2, n - 1)
