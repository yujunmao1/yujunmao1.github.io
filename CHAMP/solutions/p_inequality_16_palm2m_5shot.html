A function f(x) satisfies f(x+1)+f(x-1)=sqrt(2)*f(x). Determine whether the function is periodic, and if so, find its period.
Step-wise solution:
1. Let f(x-1)=a and f(x)=b.
2. Thus, f(x+1)=sqrt(2)*b-a, f(x+2)=b-sqrt(2)*a, f(x+3)=-a and f(x+4)=-b.

4. Thus, the function is periodic, with a period of 8.

For positive integer n, if 2^n and 5^n start with the same digit, what are possible values of this digit?
Step-wise solution:

2. Thus, we have d*10^r<2^n<(d+1)*10^r and d*10^s<5^n<(d+1)*10^s.
3. Multiplying the two inequalities we get d^2*10^(r+s)<10^n<(d+1)^2*10^(r+s).

6. We can see that d=3 is indeed possible because 2^5=32 and 5^5=3125.
7. So the answer is 3.

Step-wise solution:

3. Consider the two sequences (a^2, b^2) and (a, b) which are sorted in the same way.

5. Thus, u≥v.
6. So the smallest value of (a+b)(a^4+b^4)-(a^2+b^2)(a^3+b^3) is 0, achieved when a=b.

Find all positive integer solutions to the equation x^3+3=4y(y+1).

What is the smallest value of a/(b+c)+b/(a+c)+c/(a+b) for positive numbers a, b, c?
