A diagnoal in a convex polygon is any line connecting two non-adjacent vertices. How many diagonals are there in a convex polygon of n vertices, as an expression of n?

Below is a partial solution to the problem that may be helpful:

1. We have C(n, 2) vertex pairs, and n of them are between adjacent vertices.
To solve this problem, we need to find the total number of ways we can choose two vertices from the n vertices to form a diagonal, and then subtract the number of sides (which are lines between adjacent vertices, not diagonals).

As stated in the partial solution, there are C(n, 2) ways to choose two vertices from n vertices. C(n, 2) is the binomial coefficient, also known as "n choose 2", and it represents the number of ways to choose 2 items from n items without regard to order. The formula for C(n, k) is:

C(n, k) = n! / (k! * (n - k)!)

So for C(n, 2), we have:

C(n, 2) = n! / (2! * (n - 2)!)
          = n * (n - 1) / (2 * 1)
          = n * (n - 1) / 2

This gives us the total number of ways to choose two vertices from n vertices. However, this includes the n sides of the polygon, which are not diagonals. So we need to subtract n from this total to get the number of diagonals.

Number of diagonals = Total number of vertex pairs - Number of sides
                    = C(n, 2) - n
                    = (n * (n - 1) / 2) - n
                    = (n^2 - n) / 2 - n/1
                    = (n^2 - n - 2n) / 2
                    = (n^2 - 3n) / 2

So the expression for the number of diagonals in a convex polygon with n vertices is:

(n^2 - 3n) / 2
