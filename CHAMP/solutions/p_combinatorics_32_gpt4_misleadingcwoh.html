In how many ways can 4 horses go through the finish (with possibility of ties)?

You may find the following information useful:

1. The number of ways to choose a set of (unordered) k items from n items is C(n, k).
2. C(n, k)=n/k*C(n-1, k-1).
This problem can be solved by considering the different possible outcomes for the race. The horses can either all finish at different times, or there can be ties. 

1. All horses finish at different times: 
This is simply the number of permutations of 4 horses, which is 4! = 4*3*2*1 = 24.

2. There is a tie between two horses, and the other two horses finish at different times:
There are C(4,2) = 6 ways to choose which two horses tie. For each of these choices, there are 3! = 6 ways to order the three "groups" of horses (the tied horses count as one group). So there are 6*6 = 36 outcomes in this case.

3. There is a tie between three horses, and the other horse finishes at a different time:
There are C(4,3) = 4 ways to choose which three horses tie. For each of these choices, there are 2! = 2 ways to order the two "groups" of horses. So there are 4*2 = 8 outcomes in this case.

4. There is a tie between two pairs of horses:
There are C(4,2) = 6 ways to choose which horses make up the first pair. Then, the remaining two horses automatically make up the second pair. For each choice of pairs, there are 2! = 2 ways to order the pairs. So there are 6*2 = 12 outcomes in this case.

5. All four horses tie:
There is only 1 way this can happen, as all the horses finish at the same time.

Adding up all these possibilities, we get 24 + 36 + 8 + 12 + 1 = 81 ways for the 4 horses to go through the finish.
