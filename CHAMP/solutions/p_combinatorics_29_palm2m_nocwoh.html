For a convex polygon of n vertices, we draw all possible diagonals and suppose that no three diagonals pass through the same point. How many such intersection points do we have in the interior of the polygon, as an expression of n?
Let $n$ be the number of vertices in the polygon. For each vertex, there are $n-3$ other vertices that are not adjacent to it, so there are a total of $\binom{n}{2}(n-3)$ diagonals. However, some of these diagonals will intersect each other. To count the number of intersection points, we can first count the number of pairs of diagonals that intersect. Each diagonal intersects $n-2$ other diagonals, so there are a total of $\binom{n}{2}(n-2)$ pairs of diagonals that intersect. However, some of these pairs of diagonals intersect multiple times. Therefore, the total number of intersection points is $\binom{n}{2}(n-2)(2) = \frac{n(n-3)(2n-5)}{6}$.
