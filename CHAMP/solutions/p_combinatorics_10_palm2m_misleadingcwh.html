The number of sequences of positive integer numbers that have sum $n$ is given by the Bell number $B_n$. The Bell numbers count the number of different ways to partition a set of $n$ elements into subsets.

We can think of a sequence of positive integer numbers as a partition of the set $\\{1, 2, ..., n\\}$ into subsets, where each subset corresponds to a number in the sequence. For example, the sequence $1, 2, 3$ corresponds to the partition $\\{ \\{1\\}, \\{2\\}, \\{3\\} \\}$.

The number of ways to partition a set of $n$ elements into subsets is equal to the number of ways to choose $n-1$ elements from the set and then order them. This is because we can choose the $n-1$ elements to be the subsets, and then order them to get the sequence of numbers.

There are $n(n-1)!$ ways to choose $n-1$ elements from a set of $n$ elements. This is because for each element of the set, we have two choices: we can either include it in the subset or not include it. This gives us $2^n$ choices, but some of these choices will be the same partition, just with the elements in a different order. We can count the number of times each partition appears by considering the number of ways to order it. There are $n!$ ways to order a set of $n$ elements, so each partition appears $n!$ times.

Therefore, the number of sequences of positive integer numbers that have sum $n$ is given by the Bell number $B_n = \frac{n!}{n(n-1)!}$.
