How many sequences of positive integer numbers have sum n, as an expression of n?

Let's consider a row of n points, with n-1 gaps. We can think of these points as stars and the gaps as bars. For example, if n = 5, we would have 5 stars and 4 bars:

* | * | * | * | *

To represent a sequence of positive integer numbers that sum to n, we can place the bars in between the stars. Each bar represents a separation between two numbers. For example, if we place the bars like this:

Out of these 2n-1 objects, we need to choose the positions for the n-1 bars. This can be done in (2n-1) choose (n-1) ways, which can be written as:

Therefore, the number of sequences of positive integer numbers that sum to n is given by (2n-1)! / [n! * (n-1)!].
