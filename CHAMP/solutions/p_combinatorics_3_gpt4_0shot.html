This problem is a classic combinatorics problem with a twist. The twist is the 180-degree rotation invariance. 

Let's start with the basic problem: how many ways can we place n rooks on an n x n chess board such that no two rooks are in the same row or column? This is equivalent to asking how many permutations are there of n items, which is n!.

Now, let's consider the 180-degree rotation invariance. If we rotate the board 180 degrees, the placement of the rooks should be the same. This means that the rooks must be placed symmetrically about the center of the board. 

For odd n, the center square must contain a rook, and the remaining (n-1)/2 rooks on either side of the center must be placed symmetrically. There are (n-1)/2! ways to place these rooks. 

For even n, there is no center square, but the n/2 rooks on either side of the center line must be placed symmetrically. There are (n/2)! ways to place these rooks. 

So, the final answer is:

- For odd n: (n-1)/2! 
- For even n: (n/2)!

Note: This is assuming that rotations are considered the same placement. If rotations are considered distinct placements, then the answer would be n!.
