The number of distinct arrangements of n people around a circular table is given by (n-1)!/2.

Here's why:

1. First, we consider the rotations. If we fix one person (say, person 1), then the remaining (n-1) people can be arranged in (n-1)! ways. This is because for each of the (n-1) positions around the table, there are (n-1) choices for who to place there, then (n-2) choices for the next position, and so on, until there is only 1 choice for the last position. This gives us a total of (n-1)! arrangements.

To correct for this, we divide by n, the number of possible rotations for each arrangement. This gives us (n-1)!/n arrangements.

3. Finally, we consider reflections. If we reflect an arrangement through the center of the table, we get another arrangement that has the same neighboring relationships. For example, if we have 4 people, the arrangement [1,2,3,4] is the same as [1,4,3,2] when reflected. To correct for this, we divide by 2, the number of possible reflections for each arrangement. This gives us (n-1)!/(2n) arrangements.
