Given that a, b, c are sides of a non-degenerate 90-degree triangle with c being the hypotenuse, we know that a^2 + b^2 = c^2.

We want to find the number of values of n in {1, 2, 3, ..., 100} such that c^n > a^n + b^n.

Let's consider the cases:
