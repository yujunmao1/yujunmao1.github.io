This is a classic problem in combinatorics. The number of ways to place n rooks on an n x n chessboard such that no two are attacking each other is n!, because there are n choices for the first rook, n-1 for the second, and so on down to 1. 

However, the condition that the placement is invariant under a 180-degree rotation adds a constraint. When the board is rotated 180 degrees, each rook must land on another rook. This means that the rooks must be placed in pairs that are opposite each other with respect to the center of the board.

If n is odd, there must be one rook in the center, and the remaining n-1 rooks can be placed in (n-1)/2 pairs. If n is even, the rooks can be placed in n/2 pairs.

So, the number of ways to place the rooks is (n-1)!! if n is odd, and (n)!! if n is even, where !! denotes the double factorial, which is the product of all the integers from 1 to n that have the same parity (oddness or evenness) as n.

For example, if n=5 (odd), there are 1*3*5 = 15 ways to place the rooks. If n=6 (even), there are 2*4*6 = 48 ways to place the rooks.
