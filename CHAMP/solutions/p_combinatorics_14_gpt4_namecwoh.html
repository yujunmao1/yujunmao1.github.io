The combination formula is a way to calculate the number of possible combinations in a given set. Combinations refer to the selection of items from a larger set where the order of selection does not matter. 

The formula for combinations is often denoted as C(n, r), or "n choose r", where n is the total number of items in the set, and r is the number of items to choose from that set. 

The formula is defined as:

C(n, r) = n! / [r!(n-r)!]

Here, the "!" symbol denotes a factorial, which means to multiply a series of descending natural numbers. For example, 5! = 5*4*3*2*1 = 120.

So, if you have a set of n items and you want to know how many ways you can select r items from that set (without regard to the order of selection), you would use the combination formula. 

For example, if you have a deck of 52 cards (n=52) and you want to know how many different 5-card hands you can draw (r=5), you would calculate C(52, 5) = 52! / [5!(52-5)!] to get your answer.
