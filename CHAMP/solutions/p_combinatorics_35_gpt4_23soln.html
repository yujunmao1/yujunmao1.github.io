2n players are participating in a tennis tournament. Find the number of ways that n pairs can be constructed for the first round, as an expression of n.

Below is a partial solution to the problem that may be helpful:

1. We can order the 2n players in a row, and make the pairing (1, 2), (3, 4), (5, 6), ..., (2n-1, 2n), which can be done in (2n)! ways.
2. The different order of the two players within each pair (e.g., (1, 2) and (2, 1)) result in the same pairing.
3. Thus, we need divide the number of ways by 2^n, since we have n pairs.
4. Also, the order of the pairs does not matter. For example, the pairings ((1, 2), (3, 4)) and ((3, 4), (1, 2)) are considered the same. 

5. Therefore, we need to divide the number of ways by n! (the number of ways to arrange n pairs).

So, the number of ways that n pairs can be constructed for the first round is:

(2n)! / (2^n * n!)
