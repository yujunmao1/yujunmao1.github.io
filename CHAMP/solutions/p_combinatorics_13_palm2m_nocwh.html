1. Find the number of ways to choose a collection containing k distinguishable objects.
The number of ways to choose a collection containing $k$ distinguishable objects from a set of $n$ objects is $n$ choose $k$, which is denoted by $n \choose k$. This can be calculated using the following formula:

$$n \choose k = \frac{n!}{k!(n-k)!}$$

where $n!$ is the factorial of $n$, which is defined as $n! = n(n-1)(n-2)\dots 3 \cdot 2 \cdot 1$.

In this problem, we are given that there are $3n+1$ objects, of which $n$ are indistinguishable. This means that the remaining $3n+1-n = 2n+1$ objects are distinct. We want to choose $n$ objects from these $3n+1$ objects.

To do this, we can first choose $n$ objects from the $2n+1$ distinct objects. This can be done in $2n+1 \choose n$ ways. Then, we can choose any of the $n$ indistinguishable objects to be the $n$th object in our collection. This can be done in $n$ ways. Therefore, the total number of ways to choose $n$ objects from the $3n+1$ objects is $2n+1 \choose n \cdot n = \frac{(2n+1)!}{n!(2n+1-n)!} \cdot n = \frac{(2n+1)!}{n!n!} = \binom{2n+1}{n}$.

In conclusion, the number of ways to choose $n$ objects from $3n+1$ objects, of which $n$ are indistinguishable, is $\binom{2n+1}{n}$.
